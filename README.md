# VideoTube Backend API 🚀

A complex, production-ready backend API for a video-hosting platform (like YouTube). This project was built to master advanced backend concepts in Node.js, including complex database modeling, authentication, file handling, and aggregate pipelines.

## 📌 Features

* **Secure Authentication:** User registration, login, and logout using JWT (Access & Refresh Tokens).
* **Media Management:** Secure image and video uploading via Multer and Cloudinary.
* **User Profiles:** Update avatars, cover images, and channel details.
* **Video Functionality:** Upload, publish, edit, delete, and view videos.
* **Engagement System:** Like/dislike videos and comments.
* **Community:** Create, read, update, and delete tweets (community posts) and comments.
* **Subscription Model:** Subscribe to channels and track subscriber/subscribed counts.
* **Advanced MongoDB Aggregation:** Complex data querying for dashboards, watch history, and channel statistics.

## 🛠️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB & Mongoose
* **Authentication:** JSON Web Tokens (JWT) & bcrypt (Password Hashing)
* **File Handling:** Multer & Cloudinary
* **Other Tools:** cookie-parser, cors, dotenv
